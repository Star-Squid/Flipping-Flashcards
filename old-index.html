You are about to select a new set of flashcards.
All progress on the current set will be lost.

How it works:

  • Select a set of questions with the "Load new cards" button
  • Click the flashcard to flip to the answer side
  • Did you know the answer? Rate your knowledge: ✓ - perfect; ? - partial; ✗ - none
  • The button you choose affects how often you'll see the same flashcard
  • 3 ratings of "perfect" or 6 ratings of "partial" removes the card from rotation
