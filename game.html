How it works:

Select a set of questions from the drop-down menu or upload a custom .csv file (for example this one), or .json file (here)

Click the flashcard to flip to the answer side

Did you know the answer? Using the stars at the bottom, rate your knowledge: ☆ - none; ☆☆ - partial; ☆☆☆ - perfect

The rating you select affects how often you'll see the same flashcard

3 ratings of "perfect" (3 stars) or 6 ratings of "partial" removes the card from rotation in the current session

Change the flashcard set at any time by clicking the refresh button
